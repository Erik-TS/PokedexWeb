
# PokedexWeb

A website that uses the [PokéAPI](https://pokeapi.co) to search and show information about the Pokémon.

## Used Stack

**Front-end:** React, NextJS, Bootstrap, ChartJS

**Back-end:** Node

**Language:** Typescript
